Intro to Causality

Causal inference tries to understand how and when causal conclusions can be drawn from data.

The main modelling tool is a Structural Causal Model (SCM) which describes the complete system of interacting variables

The relationships between the variables can be represented via a Causal Bayesian Network (CBN).

The main feature of SCM with respect to MDP is that is allows explicit reasoning about counterfactuals, that is what could have happened.

This is of particular interest for RL since it shifts the reasoning from what an agent can do to, as in standard MDP, to what an agent could have done.

Counterfactual reasoning is done via the concept of intervention, which corresponds to changing some of the structural assignments in the SCM, namely

Causal RL and Counterfactual Credit Assignment

There are several good results by including causal reasoning in RL, some examples are:

Transportability and Data Fusion

By making assumptions about the data-generating system and reasoning about causal relationships between variables of interest, it is possible to alleviate issues such as distributional shift, inductive bias and so on, by algorithmically combining datasets collected under different conditions and policies.

Moreover, a causal model of the underlying system allows for establishing bounds on performance, while also providing techniques for selecting optimal actions LB-2018.

Counterfactual Reasoning

The abillity to reasoning in terms of counterfactuals is essential in problems where sampling is expensive, safety critical or impossible. Improving the data-effciency of learning requires full use of available information, such as causal structures.

These intuitions led to MDP with Unobserved Confounders, which model the setting where there are latent external factors which confound for states, actions and rewards, potentially causing dependence which can harm learning performances.

Finally, a counterfactual approach is useful for directing exporation towards areas of the system that have causal unknowns and for optimizing chosen actions (interventions) withing the causal model.

Causal MARL

Some have already tried to improve performance on tasks where credit assignment is challenging by using counterfactual information, as in . By communicating counterfactuals as well, the performances can be improved too VVMplus-2020. JLHplus-2018 proposed to used causality as an intrinsic motivation via social influence.

Finally, causal reasoning can be extremely effortful with respect to MADRL, where multiple agents come with inherently scalability benefits due to decentralization of learning and/or execution.

GSE-2020 extend the idea of Bayesian Game formulation by including a causal model about the environment, leading to a Bayesian framework where agents hold probabilistic beliefs about possible causal models.

Nonetheless, all these works actually considered a Level 2 reasoning about interventions and not actual counterfactuals.

In order to extend this reasoning, Multi-Agent Causal Models (MACM) were proposed as in MMML-2005, which extend causal reasoning with respect to MAs, were actions are taken with respect to private and public variables. Tackling non-stationarity

Common approaches to tackling non stationarity include centralizing the training procedure SFZplus-2021, accounting for other agents RDSF-2018 and applying communication protocols.

Causal inference offers powerful methods over these pathology, especially when agents have structural knowledge of how uncertainties are arising.

Knowledge sharing

An effective way to increase cooperation in MAs is by using communication methods, which are designed to boost learning performance by exploiting existing knowledge encoded in other agents. Other sorts of sharing can be seen as transfer and imitation, as for emergent behaviour studies of WDLZ-2020.

Generalized Off-policy MARL

Distributional shifts problems are particularly exacerbated in MAs, where each agent have hidden decision policies and influences the response of the environment YMLplus-2021, PCRA-2019

Model Learning
